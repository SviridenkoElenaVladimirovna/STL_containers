Detailed Description

template<typename T>
class CyclicalList< T >

A cyclic bilinked list with an STL container interface.

Stores elements of type T as a cyclic doubly-linked structure. Supports insertion at start and end, deletion, iterators, and relocation.

Template Parameters
TThe type of elements stored in the list. Must be copyable and destructible.
